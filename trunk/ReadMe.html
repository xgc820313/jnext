ReadMe file for JNEXT 1.0.5 sources

The following is an explanation of the source tree for JNEXT 1.0.5

Projects

PathProject Description
src\framework-ActiveXWindows VS2005 project for JNEXT as ActiveX component for IE
src\framework-npruntime Linux Makefile and Windows VS2005 project for JNEXT as NPAPI plugin
src\plugins\filereader Linux Makefile and Windows VS2005 project for filereader sample extension. This is a self contained simple example that illustrates how to create your own Javascript native extension using the JNEXT plugin framework.
src\plugins\socketsLinux Makefile and Windows VS2005 project for the Sockets JNEXT plugin that enables using TCP sockets from Javascript. Compling this project requires the PWLib library which can be downloaded from http://www.voxgratia.org
